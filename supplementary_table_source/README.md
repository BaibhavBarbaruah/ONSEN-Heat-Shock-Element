# Supplementary-table source sheets

This directory contains tab-separated source sheets for the final Biology Open supplementary Tables S1-S14. The files preserve the journal numbering and worksheet contents without committing binary Excel files.

Run the complete workflow (`source("00_run_pipeline.R")`) or run `03B_threshold_and_continuous_sensitivity.R` followed by `08_write_supplementary_tables.R` to create `Table_S1.xlsx` through `Table_S14.xlsx` under `ONSEN_OUTPUT_ROOT`.

Table S8 contains a large region-level sheet. Its compact summary and QC sheets are deposited here; the complete `S8C_region_metrics` sheet is regenerated by `03B_threshold_and_continuous_sensitivity.R` and inserted during workbook construction.

Final-audit clarifications:

- Table S4 reports 5,119 alternative exact-GC comparator sequences; the 5,120th sequence is the submitted design itself.
- Table S9 includes `S9D_selected_outliers`, the exact eight-region HSF-rich non-ONSEN subset used for Figs 4B-D and 5C.
- Table S10 uses a single 0.05 CPM pseudocount for reported class- and window-level log2 fold changes.
