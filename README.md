# Representative Fig. 7D motif profiles

This directory contains the two small JASPAR PFMs used by the reproducible Fig. 7D logo workflow:

- `MA1667.2_HSFC1.jaspar` — HSF-family representative, HSFC1
- `MA0981.2_DOF1.8.jaspar` — DOF-family representative, DOF1.8

The matrices are redistributed from JASPAR under the JASPAR/Creative Commons attribution terms. JASPAR 2026 should be cited when these profiles are used. The profiles describe sequence preferences; they do not establish transcription-factor occupancy in vivo.
